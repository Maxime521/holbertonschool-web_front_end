# Responsive Design

## Description
Projet Responsive Design réalisé dans le cadre de la formation **Holberton School**.  
L’objectif est de créer une page web qui s’adapte à toutes les tailles d’écran (mobile, tablette, desktop) grâce au **responsive design** et aux bonnes pratiques CSS.

## Technologies
- HTML
- CSS

## Installation
1. Clonez ce dépôt :
```bash
git clone https://github.com/Maxime521/holbertonschool-web_front_end.git
```
2. Ouvrir le fichier index.html dans votre navigateur :
```bash
open index.html
```
ou double-cliquer sur le fichier dans votre explorateur de fichiers.

## Usage
Ouvrez la page dans votre navigateur et redimensionnez la fenêtre pour voir l’adaptation du layout selon différentes tailles d’écran.
